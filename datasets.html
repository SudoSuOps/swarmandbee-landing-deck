Every pair passes 6 deterministic quality gates before earning platinum status. No LLM judges -- deterministic code that can't be gamed.

Valid JSONL, required fields present

GATE 2

Length Gate

Answer meets minimum depth threshold

GATE 3

Duplication Gate

MD5 fingerprint dedup across all shards

GATE 4

Specialty Gate

Verified against domain taxonomy

GATE 5

Coherence Gate

Question-answer alignment scoring

GATE 6

Pairs that pass all gates enter Chain of Verification -- Llama-70B rewrites for clarity, Qwen-235B scores on 5 criteria. Minimum 20/25 to earn platinum.
