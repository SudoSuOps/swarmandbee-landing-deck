Quick Start

Get from zero to your first API call in under 60 seconds.

STEP 1

Get Your Key

Subscribe at /pricing or sign up for the newsletter to get 10 free credits and an sb_live_ API key.

STEP 2

Set Your Header

Pass your key via X-API-Key header or Authorization: Bearer on every request.

STEP 3

Make a Call

Hit any endpoint. Credits deducted per call. Check balance anytime at /wallet/balance.

Your first API call
curl -X POST https://api.swarmandbee.ai/v1/chat/completions \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "model": "swarm-atlas-27b", "messages": [ {"role": "system", "content": "You are a CRE capital markets analyst."}, {"role": "user", "content": "Underwrite a $14.4M industrial warehouse in Memphis, 6.2% cap, 120K SF, NNN lease."} ] }'
OpenAI-compatible. Drop in your existing OpenAI client code. Just change the base URL to api.swarmandbee.ai and swap your key.
Python (OpenAI SDK)
# pip install openai from openai import OpenAI client = OpenAI( base_url="https://api.swarmandbee.ai/v1", api_key="YOUR_API_KEY" ) response = client.chat.completions.create( model="swarm-atlas-27b", messages=[ {"role": "system", "content": "You are a CRE capital markets analyst."}, {"role": "user", "content": "Analyze cap rate compression in DFW industrial."} ] ) print(response.choices[0].message.content)
JavaScript (fetch)
const response = await fetch("https://api.swarmandbee.ai/v1/chat/completions", { method: "POST", headers: { "Authorization": `Bearer ${API_KEY}`, "Content-Type": "application/json" }, body: JSON.stringify({ model: "swarm-atlas-27b", messages: [ { role: "system", content: "You are a CRE capital markets analyst." }, { role: "user", content: "What's the refinancing risk on a 110% LTV CMBS loan?" } ] }) }); const data = await response.json(); console.log(data.choices[0].message.content);

Authentication

Every metered request requires an API key. Pass it in one of two ways:

# Option 1: X-API-Key header curl -H "X-API-Key: sb_live_your_key_here" https://api.swarmandbee.ai/wallet/balance # Option 2: Bearer token curl -H "Authorization: Bearer sb_live_your_key_here" https://api.swarmandbee.ai/wallet/balance
Key prefixes: sb_live_ for free-tier keys (newsletter signup). sk_swarm_ for paid data marketplace keys. Both work on all endpoints.
Response Headers

Every metered response includes credit tracking headers:

X-Credits-Used: 2 X-Credits-Remaining: 48

Training Data API

Pull verified training pairs programmatically. Filter by vertical, specialty, difficulty, and quality tier. JSONL format.

Pull CRE training data
curl "https://api.swarmandbee.ai/api/data/pull?vertical=cre&specialty=underwriting&tier=platinum&limit=100" \ -H "Authorization: Bearer YOUR_KEY"
Browse the catalog (free, no auth)
curl https://api.swarmandbee.ai/api/data/catalog
